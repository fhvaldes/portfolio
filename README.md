# Fredy Hernández | Portfolio

Portfolio interactivo profesional — Backend Developer · Python · Django · FastAPI · AI/LLM Integration

## Descripción

Portfolio personal desarrollado con React, TypeScript y Tailwind CSS. Inuye información sobre experiencia profesional, proyectos destacados en GitHub, stack tecnológico y terminal interactiva.

## Stack Tecnológico

- **Frontend:** React 19, TypeScript, Tailwind CSS, Vite
- **Animaciones:** Motion (Framer Motion)
- **Iconos:** Lucide React
- **Servidor:** Express.js (desarrollo con Vite middleware)

## Ejecutar Localmente

**Prerrequisitos:** Node.js 20+

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abrir en el navegador:
   ```
   http://localhost:3000
   ```

## Scripts Disponibles

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Servidor de desarrollo con HMR |
| `npm run build` | Build de producción |
| `npm run start` | Ejecutar build de producción |
| `npm run lint` | Verificación TypeScript |
