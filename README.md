# Fredy Hernández | Portfolio

Portfolio interactivo profesional — Python Backend Developer, IA, LLMs & RAG.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `GEMINI_API_KEY` in [.env.local](.env.local) with your Google Gemini API key
3. Run the app:
   `npm run dev`
